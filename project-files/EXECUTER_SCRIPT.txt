❓ PROBLÈME : "psql : command not found"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Solution : Ajoutez PostgreSQL au PATH

1. Trouvez le dossier d'installation de PostgreSQL :
   Exemple : C:\Program Files\PostgreSQL\16\bin

2. Ajoutez-le au PATH :
   - Recherchez "Variables d'environnement" dans Windows
   - Cliquez sur "Variables d'environnement"
   - Sélectionnez "Path" → Modifier
   - Ajoutez : C:\Program Files\PostgreSQL\XX\bin
   - OK → OK → Redémarrez PowerShell

OU utilisez le chemin complet :
"C:\Program Files\PostgreSQL\16\bin\psql" -U postgres -d cosone_db -f insert_centres_from_csv.sql
